Algorithm Journal: Graph Colouring with k=2

You’re given an array of rivals: rivals = ["A", "B", "C"] And a list of matches between two rivals represented in tuple form: matches = [("A", "B), ("B", "C")] Assign a label (either “good” or “bad”) to each rival, so that there is no match where both rivals have the same label. Initial thoughts on the solution At a first glance, it sounds very similar to the Graph Colouring problem. In fact I believe it is no different from it, although it is a special case of the graph colouring problem with 2 as the number of colour.
